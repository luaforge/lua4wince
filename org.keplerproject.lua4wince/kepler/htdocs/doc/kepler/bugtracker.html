Kepler Bug Trackers

Kepler consists of a set of components. If you have a bug to report in one module please use the corresponding bug tracker on LuaForge:

  • CGILua
  • LuaBinaries
  • LuaExpat
  • LuaFileSystem
  • LuaLogging
  • LuaSQL
  • LuaZip
  • Xavante

If you don't know which module is involved in the bug, or if the bug is on the Kepler documentation or in the installer please use the Kepler bug tracker on LuaForge.

For more information please contact us. Comments are welcome!

You can also reach other Kepler developers and users on the Kepler Project mailing list.
