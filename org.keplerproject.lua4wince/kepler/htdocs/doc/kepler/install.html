Kepler Installation Instructions

Kepler can be installed on Windows, Unix, Mac OS X and other platforms.

To install Kepler on an Unix/OS X platform you'll have to use the Unix Package (a tar.gz or zip archive with the source files) and have some notion of Unix commands and packages installation from the source.

To install Kepler on a Windows platform you just have to use the Kepler installer (a Windows executable file) and follow the instructions.

To install Kepler on other platforms you can use the Unix Package and adapt the makefiles to the target platform.

After you install Kepler you'll need to check how to use Kepler with your Web server. For more details please refer to the installation documentation for the desired platform:

  • Unix/OS X
  • Windows

The installation procedures can have some subtleties and this documentation will be constantly updated to reflect this. If you have any questions concerning the installation process or some platform detail, please contact us.

You can also reach other Kepler developers and users on the Kepler Project mailing list.
