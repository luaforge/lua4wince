Overview

Kepler is a Lua web development platform that offers the same advantages of Lua. Kepler is simple, portable, light, embeddable and extensible

The Kepler Platform consists in the Windows, Unix and OSX distribution of the components that allow the creation of Web Applications and the use of SQL, XML, Zip, MD5 etc.

Kepler is free software and uses the same license as Lua 5.1.

More information about the project can be found at the Kepler FAQ.

Status

Current version is 1.1 Beta 2. It was developed for Lua 5.1.

Download

Kepler source can be downloaded from its Lua Forge page. A Windows installer for Kepler (using LuaBinaries) can also be found at the same LuaForge page.
